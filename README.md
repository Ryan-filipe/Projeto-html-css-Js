# Análise de Números

Este projeto é um analisador de números em JavaScript, HTML e CSS, inspirado em interfaces modernas como Amazon. Ele permite adicionar números de 1 a 100, exibir a lista de números, calcular maior, menor, soma e média, e limpar a lista.

## Funcionalidades

- Adicionar números de 1 a 100
- Evitar números duplicados
- Exibir lista de números adicionados
- Calcular
  - Total de valores
  - Maior valor
  - Menor valor
  - Soma
  - Média
- Limpar lista e resultados
- Mensagens de erro animadas
- Design responsivo

## Tecnologias

- HTML5
- CSS3 (flexbox, gradientes, responsividade)
- JavaScript (ES6)

## Design

- Inspirado em interfaces corporativas modernas
- Gradiente azul no fundo, cartão branco centralizado
- Botões com cores destacadas (adicionar = laranja, limpar = vermelho)
- Responsivo para mobile
